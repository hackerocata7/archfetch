# Archfetch
A simple fetch app made in ***Rust*** (A very original idea, I know).
## How to set this up
 1. Run `git clone https://github.com/hackerocata7/archfetch.git`.
 2. Copy `src/logo.txt` to `~/archfetch/logo.txt`.
 3.  In the directory, run `cargo build --release`.
 4. Copy `archfetch/target/release/archfetch` to `/usr/local/bin`.
 5.  In terminal, run `archfetch`.
